**Delay Test**
1. Show group of objects with different colors
2. Delay (with fMRI)
3. Test

- Increase in blood flow in Intraparietal Sulcus (dorsal stream) when there's nothing on display!
- Activation is correlated with memory load
- **Maximum** memory load is 3-4 items (as opposed to 7 in past literature)
- CDA: Contralateral Delay Activity, gold standard of memory activation if you see the pattern in the graphs

**Object Search**

Past studies found no evidence of working memory in visual search, even though it makes sense (we don't sample without replacement)

The test here is to show multiple figures in various orientations and ask if the 180 degrees one is present.

 In search the information has to be on the screen continuously so we can't detect CDA, instead find something close enough and call it CSA

We know that amplitude of CDA correlate with number of items kept in VWM (k).
We find out that people only keep a few items (3-4) in working memory in search, just like in detection, by the amplitudes of the CSA!

- CSA starts later than CDA
- Time to reach peak CSA happens later for people with high working memory capacity
